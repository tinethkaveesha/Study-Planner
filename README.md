# Study Planner 1.0.2

A modern web application designed to help students organize, plan, and track their study schedules efficiently.

## Tech Stack

- **Frontend:** React.js/ HTML/ JS
- **Styling:** CSS3 / Tailwind CSS
- **Package Manager:** npm / yarn
- **Node.js:** Version 14.0 or higher
- **Build Tool:** Vite
- **State Management:** Context API
- **Database:** Firebase

## Setup Instructions

### Prerequisites
- Node.js (v14.0 or higher)
- npm or yarn package manager
- Git

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/tinethkaveesha/Study-Planner
   cd Study-Planner
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   yarn install
   ```

3. **Configure environment variables**
   - Create a `.env` file in the root directory
   - Add necessary environment variables (API endpoints, keys, etc.)

4. **Start the development server**
   ```bash
   npm start
   # or
   yarn start
   ```

5. **Open in browser**
   - Navigate to `http://localhost:3000`

### Build for Production
```bash
npm run build
# or
yarn build
```
